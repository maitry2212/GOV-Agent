

### **GOV-Agent: AI Government Services Assistant** | *Full-Stack AI Project*

* Built a **multi-agent AI system** using LangGraph to provide accurate guidance from official sources (.gov.in, .nic.in)
* Designed a **7-node workflow pipeline** for query processing, research, and verification
* Developed **FastAPI backend** using Groq (LLaMA-3.3-70B) + Tavily for reliable responses
* Implemented **JWT authentication, persistent chat, and async PostgreSQL (Neon)** queries for scalable, low-latency performance
* Deployed on **Render (backend)** and **Vercel (frontend)**

**Tech:** FastAPI, LangGraph, Groq, Tavily, React, Tailwind, PostgreSQL, SQLAlchemy, JWT
